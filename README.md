# SERP Scraper
Get rankings from Search Engines (Google)
## Dependencies
* [mechanize](https://github.com/sparklemotion/mechanize)
* [nokogiri](https://github.com/sparklemotion/nokogiri)
* [addressable/uri](https://github.com/sporkmonger/addressable)
